Caesar Cipher
Shifts every letter by a fixed amount K. One of the oldest and simplest encryption techniques.
SubstitutionMonoalphabeticSymmetric
Output
—
Encrypt: C = (P + K) mod 26  ·  Decrypt: P = (C − K + 26) mod 26
Vigenère Cipher
Uses a repeating keyword to shift each letter by different amounts — a polyalphabetic substitution.
SubstitutionPolyalphabetic
Output
—
Encrypt: Cᵢ = (Pᵢ + Kᵢ) mod 26  ·  Decrypt: Pᵢ = (Cᵢ − Kᵢ + 26) mod 26
Hill Cipher
Encrypts pairs of letters using matrix multiplication in modular arithmetic mod 26.
SubstitutionPolygraphicMatrix-based
Output
—
Encrypt: C = K · P mod 26  ·  Decrypt: P = K⁻¹ · C mod 26  ·  Matrix must be invertible mod 26
Playfair Cipher

DES — Data Encryption Standard
Symmetric-key block cipher using a 56-bit key and 16 Feistel rounds on 64-bit blocks. Educational simulation.
SymmetricBlock Cipher64-bit BlockFeistel
Output (hex)
—
Structure: Initial Permutation (IP) → 16 Feistel Rounds: Expand(R) + XOR(Kᵢ) + S-Boxes + Permute + Swap → Final Permutation (IP⁻¹)
AES — Advanced Encryption Standard
Block cipher with 128/192/256-bit keys operating on a 4×4 byte state matrix. Educational simulation.
SymmetricBlock Cipher128-bit BlockRijndael
Output (hex)
—
Each round: SubBytes → ShiftRows → MixColumns → AddRoundKey  ·  Final round skips MixColumns
RSA Algorithm
Public-key cryptosystem. Encrypt with the public key (e, n), decrypt with the private key (d, n).
AsymmetricPublic KeyNumber Theory
Output
—
Key gen: n = p·q, φ(n) = (p−1)(q−1), choose e where gcd(e,φ)=1, d = e⁻¹ mod φ  ·  Enc: C = Mᵉ mod n  ·  Dec: M = Cᵈ mod n
Diffie-Hellman Key Exchange
Two parties derive the same shared secret over an insecure channel without ever transmitting the secret itself.
Key ExchangeDiscrete Log
Alice sends: A = gᵃ mod p  ·  Bob sends: B = gᵇ mod p  ·  Shared secret: s = Bᵃ mod p = Aᵇ mod p
ElGamal Cryptography
Probabilistic public-key encryption based on the Diffie-Hellman problem. Ciphertext is a pair (C₁, C₂).
AsymmetricProbabilisticDiscrete Log
Output
—
Key gen: y = gˣ mod p  ·  Encrypt: C₁=gᵏ mod p, C₂=M·yᵏ mod p  ·  Decrypt: M = C₂·(C₁ˣ)⁻¹ mod p
Elliptic Curve Cryptography (ECDH)
Uses algebraic structure of elliptic curves over finite fields for key exchange. y² = x³ + ax + b (mod p)
AsymmetricElliptic CurveECDH
Output
—
Point multiply: PA = nA·G, PB = nB·G  ·  Shared secret: nA·PB = nB·PA  ·  Point add/double uses modular inverse
